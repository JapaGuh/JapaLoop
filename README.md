# All Clicking Macro

A Python application featuring a modern, responsive user interface built with Flet, featuring integrated icon support and multi-threaded execution.

## Project Contents

This repository contains the core files for AllClickingMacro:

*   **Python Source Code** - The main application logic written in Python using the Flet framework.
*   **Asset Folder** - Custom fonts (Montserrat) and branding assets used to enhance the UI.
*   **Application Icon** - A custom `.ico` asset used for the window and taskbar identity.

## About This Project

AllClickingMacro has been instrumental in my learning journey as a developer. Through building this project, I gained valuable experience in:

*   **Python Programming** - Mastering logic, loops, and external library integration.
*   **Asynchronous Execution** - Implementing `threading` to keep the UI responsive during macro execution.
*   **UI/UX Design with Flet** - Creating dynamic layouts, input validation, and real-time visual feedback.
*   **Input Simulation** - Using `pyautogui` and `keyboard` for low-level system interaction and hotkey detection.

This project represents an important step in my programming education and demonstrates practical application of modern desktop development concepts.

## Features

*   **Dynamic UI**: Window automatically resizes when custom options are selected.
*   **Smart Validation**: Real-time checking for valid keys and delay values.
*   **Safety Lock**: Visual feedback with lock icons and disabled fields while running.
*   **Multi-Unit Support**: Toggle between seconds and minutes with automatic conversion.
